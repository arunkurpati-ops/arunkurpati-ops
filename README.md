## 💼 Featured Project

### 🏥 [Healthcare Readmission AI](https://github.com/arunkurpati-ops/healthcare-readmission-ai)

**AI/ML-powered 30-Day Hospital Readmission Risk Prediction System**

A comprehensive healthcare analytics project that:
- **Predicts readmission risk** using EHR data and clinical features
- **Identifies high-risk patients** for targeted interventions
- **Implements interpretable ML models** (Logistic Regression, Random Forest, XGBoost)
- **Provides explainability** using SHAP for clinical decision support
- **Ensures compliance** with HIPAA, GDPR, and ethical AI guidelines

**Key Technologies**:
- `Python` | `Pandas` | `Scikit-learn` | `XGBoost` | `SHAP`
- `Flask/FastAPI` | `Streamlit` | `Jupyter`

**Impact**: Potential to prevent 40-50% of avoidable readmissions with targeted interventions.
